# roms

Drop your Game Boy (`.gb`), Game Boy Color (`.gbc`), and Game Boy Advance
(`.gba`) ROMs in this folder. Running `termboy` with no arguments opens a
picker that scans here, grouped by hardware.

Everything in this folder **except this README is git-ignored** — ROMs and the
`.sav` battery files termboy writes next to them are never committed. Bring your
own legally obtained ROMs.
